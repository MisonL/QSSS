# 量化交易策略系统

## 项目简介

这是一个基于Python的量化交易策略系统，通过多维度技术分析和机器学习模型，为A股市场提供选股策略建议。系统具备动态资源管理能力，可以自适应调整运行参数，提供包括超短线交易在内的多种分析维度。

## 主要功能

- 🔍 全市场股票数据自动获取和分析
- 🤖 机器学习模型预测股票趋势
- 📊 多维度技术指标分析
- ⚡ 超短线交易策略分析
- 🔄 动态多线程管理
- 📈 实时性能监控

## 技术特点

- 智能数据缓存机制
- 自适应系统资源管理
- 多数据源容错机制
- 异常检测和自动处理
- 完整的中文界面

## 快速开始

1. 确保系统满足以下要求：
    - Python 3.8+
    - 4核及以上CPU
    - 8GB及以上内存
    - 1GB可用硬盘空间

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 运行程序：
   ```bash
   python main.py
   ```
